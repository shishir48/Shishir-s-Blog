# Shishir-s-Blog

A simple React blog built with Vite.

## Pages

- `/` — Landing page with information about yourself.
- `/blogs` — Blog posts page.

## Getting started

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
